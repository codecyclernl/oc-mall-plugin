# AddressSelector

The AddressSelector component displays an address that belongs to the user's cart. It gives the user the ability to edit it or select a different address.

This component is part of the Checkout component and is used to display the currently active billing and shipping Address on the checkout page.

# Properties

# type (string)

Display the billing or shipping address of the user's cart.

# Example implementations

TIP

This component is part of the Checkout component. You do not have to implement it separately.

# Display the address selector

[addressSelector shippingAddress]
type = "shipping"

[addressSelector billingAddress]
type = "billing"
